# FranciscoSolis Website

This repository hosts all the files for the frontend site of [FranciscoSolis](https://franciscosolis.cl). 
It uses [bun.sh](https://bun.sh) as toolchain and package manager, and [vite](https://vitejs.dev).

# Commands
- `bun dev` - Start dev server
- `bun build` - Build for production
- `bun preview` - Serve production build
- `bun lint` - Run ESLint
- `bun format` - Run Prettier
